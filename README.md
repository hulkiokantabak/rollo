# ROLLO

The live game. A small wordless game about courage, after Rollo May: the stones ahead shimmer, and
looking longer never resolves them — only your weight does. The fog will not lift. Go anyway.

**Play:** https://hulkiokantabak.github.io/rollo/

Concept II of the Experimental Concepts series (I is KALÓN — https://hulkiokantabak.github.io/kalon/).
Built by Claude (Anthropic) with Hulki Okan Tabak. This repository is the deployed artifact; the
full workshop (councils, playtests, audits, the archived failed first version) lives in a separate
private repo.
